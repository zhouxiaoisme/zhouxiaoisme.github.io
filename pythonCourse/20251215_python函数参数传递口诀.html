Python函数参数传递口诀

针对技工院校学生设计的Python函数参数传递方式记忆口诀，通过简单易懂的比喻和代码示例，帮助学生快速理解和掌握。

位置参数

顺序要对，个数要齐。

核心特点

函数调用时，实参的顺序和个数必须与函数定义时的形参完全一致，就像流水线上的零件必须按固定顺序放置。

代码示例

def 加工零件(零件名, 加工时间, 转速):
    print(f"加工{零件名}，用时{加工时间}分钟，转速{转速}转/分")

# 正确：参数顺序、个数完全匹配
加工零件("齿轮", 10, 3000)

# 错误：顺序或个数不对会导致错误
# 加工零件("齿轮", 3000, 10) # 顺序错误
# 加工零件("齿轮", 10) # 个数不对

默认参数

定义时给备胎，调用时可省力。

核心特点

在定义函数时给形参一个默认值。调用时如果不提供该参数，就使用默认值。就像机器有默认设置，不调就用默认，要调就改一下。

代码示例

def 启动设备(设备名, 速度=1500, 模式="自动"):
    print(f"启动{设备名}，速度{速度}，模式{模式}")

# 使用默认值
启动设备("CNC机床") # 速度=1500，模式="自动"

# 修改部分默认值
启动设备("铣床", 2000) # 速度=2000，模式="自动"

# 修改所有参数
启动设备("钻床", 1200, "手动")

关键字参数

点名道姓，不怕乱序。

核心特点

调用函数时，通过"形参名=实参值"的方式指定数据，顺序可以打乱。就像设备控制面板上的按钮，按哪个调哪个，不用管顺序。

代码示例

def 设置工艺参数(温度, 压力, 时间):
    print(f"温度:{温度}°C, 压力:{压力}MPa, 时间:{时间}分钟")

# 传统位置参数方式
设置工艺参数(200, 5, 30)

# 关键字参数方式（顺序可打乱）
设置工艺参数(时间=30, 温度=200, 压力=5)

# 混合使用（位置参数必须在关键字参数前面）
设置工艺参数(200, 时间=30, 压力=5)

可变参数

一个星号*，多少都行，打包成元组。

核心特点

在形参前加*，接收任意数量的位置参数，打包成元组。用于处理事先不知道需要多少个参数的情况，就像零件收集箱，放多少零件都行。

代码示例

def 计算总重量(*零件重量):
    总重量 = 0
    for 重量 in 零件重量:
        总重量 += 重量
    print(f"零件总重量: {总重量}kg")

# 可以接收任意数量的参数
计算总重量(5.2, 3.1, 7.8) # 3个参数
计算总重量(2.5, 4.3) # 2个参数
计算总重量(10) # 1个参数
计算总重量() # 0个参数（空元组）

四种参数传递方式对比

参数类型 口诀 关键特点 使用场景 类比
位置参数 顺序要对，个数要齐 实参顺序、个数必须与形参完全一致 简单函数，参数少且顺序明确 流水线工位
默认参数 定义时给备胎，调用时可省力 定义时设置默认值，调用时可省略 参数有常用默认值的情况 机器默认设置
关键字参数 点名道姓，不怕乱序 调用时用"形参名=实参值"，顺序可打乱 参数多、易混淆顺序时 控制面板按钮
可变参数 一个星号*，多少都行，打包成元组 参数个数可变，接收为元组 参数数量不确定时 零件收集箱

教学提示：建议让学生先掌握位置参数，然后逐步学习其他参数类型，并通过实际编程练习加深理解。
